# Google DevFest Workshop - Chrome DevTools MCP Server

这是一个关于 Chrome DevTools MCP Server 的技术工作坊网站，由 GDG 社区组织举办。

## 🌟 关于这个项目

这个项目展示了一个现代、简洁的网站，介绍了 Google DevFest Workshop 的相关内容，包括：

- **首页**: 总体信息展示和工作坊介绍
- **GDG & DevFest**: 介绍 Google Developer Groups 和 DevFest 活动
- **DevTools MCP**: 详细介绍 Chrome DevTools MCP Server 技术
- **关于我**: 工作坊组织者的个人信息

## 🚀 技术特点

- 现代简约的设计风格，参考百度风格
- 响应式布局，适配各种设备
- 优雅的渐变色彩和动画效果
- 清晰的导航结构和用户体验
